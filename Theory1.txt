Основные методы для работы с коллекциями. Классы Collections, Arrays.

классы collections, arrays являются частью general packages. доступны в пакете java.util.
в целом они предназначены для облегчения выполнения типовых операций с коллекциями и массивами (сортировка, поиск, стандартные преобразования и т.д.).

класс collections содержит статические методы для работы с коллекциями. (List, Set, Map и т.д.)

- sort(List<T> list) - сортирует список в естественном порядке. также есть перегруженная версия, принимающая компаратор вторым аргументом для кастомной сортировки (сложность вроде линеарифмическая т.к. это обычная быстрая сортировка хоара, но не уверен).
- binarySearch(List<? extends Comparable<? super T>> list, T key) - бинарный поиск в списке с логарифмическим временем (список должен быть предварительно посорчен).
- reverse(List<?> list) - разворачивает список в обратном порядке.
- shuffle(List<?> list) - перемешивает элементы в случайном порядке.
- min/max(Collection<? extends T> coll) - возвращает минимальный/максимальный элемент.
- fill(List<? super T> list, T obj) - заполняет список указанным значением.
- copy(List<? super T> dest, List<? extends T> src) - копирует элементы из одного списка в другой.
- synchronizedList(List<T> list)/synchronizedSet(Set<T> set): возвращает потокобезопасный враппер для списка/множества.

класс arrays предоставляет методы для работы с массивами.

- sort(T[] a) - сортирует массив.
- binarySearch(T[] a, T key) - бинарный поиск.
- fill(T[] a, T val) - заполняет массив указанным значением. есть перегруженная версия для заполнения конкретного слайса.
- equals(T[] a, T[] b) - проверяет два массива на равенство.
- toString(T[] a): преобразует массив в список
- asList(T... a): преобразует массив в список
- stream(T[] array): преобразует массив в поток stream api.
