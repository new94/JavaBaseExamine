Файлы в Java. Основные потоки чтения/записи. Основные методы работы с потоками чтения/записи.

работа с файлами происходит через потоки ввода/вывода (I/O). также существует новая версия - NIO (new input output).

типы потоков:
- байтовые потоки - используются для работы с байтами (двоичными данным). например, изображения, аудио, .bin.
- символьные потоки - используются для работы с текстовыми данными.

байтовые потоки:
- InputStream - абстрактный класс для чтения байтов (FileInputStream, ByteArrayInputStream).
- OutputStream - абстрактный класс для записи байтов (FileOutputStream, ByteArrayOutputStream).

символьные потоки:
- Reader - абстрактный класс для чтения символов (FileReader, BufferedReader, InputStreamReader).
- Writer - абстрактный класс для записи символов (FileWriter, BufferedWriter, OutputStreamWriter).

основные методы:
    -InputStream/OutputStream:
        - read - читает массив байтов или один байт.
        - write — записывает массив байтов или один байт.
        - close — закрывает поток, освобождая ресурсы.
    - Reader/Writer:
        - read - читает массив символов или один символ.
        - readLine — читает строку текста.
        - write — записывает массив символов, один символ или строку.
        - close — закрывает поток.

с потоками желательно работать, используя try-with-resources для их автоматического закрытия.

также есть специфические классы, например ObjectInputStream/Object (для записи обьектов), Scanner (удобен для парсинга примитивных типов из потока) или RandomAccessFile (для произвольного перемещения по файлу через метод seek).
